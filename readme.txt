Технические реализации системы продажи напитков.

[Пользователь]:
+ Напитки представлены названием, картинкой, стоимостью
+ Вносить монеты номиналом 1, 2, 5, 10
+ Заблокировать использование "блокированных" монет
+ Показывать внесенную сумму
+ Покупка напитков, уменьшать кол-во напитков при покупке
+ Возврат монет в виде сдачи (допустимо использовать заблокированные монеты)

[Администратор]:
+ Доступ к панели администратора по секретному ключу
+ Добавление нового напитка
+ Удаление напитка
+ Редактирование напитка: кол-во, стоимость, изображение
+ Возможность блокировки монет
+ Возможность изменять кол-во монет в автомате

[Дополнительно]:
+ Возможность импорта напитков
+ Возможность покупки нескольких напитков перед получением сдачи
